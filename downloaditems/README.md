# 校内下载站前端

## 支持链接校园网后直接下载，不用登录点击即下
## 软件需管理员登录后审核通过才可以正常上传，管理员可以登陆后查看并管理软件包括但不限于上传，描述，删除软件等
## 提供热门软件排名，按照学院来进行划分
## 学生用户只有查看软件详情并下载的权限，没有上传软件的权限
## 管理员登录可以使用密码和验证码登录，也可以利用校内下发的安全密钥进行登录
